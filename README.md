# ヤフオク落札価格チェッカー

指定したヤフオク出品者の過去の落札商品について、特定の商品名で絞り込んで落札価格の履歴を確認できるWebアプリケーション

## 機能

- 出品者の評価ページURLから過去の落札商品を全て取得
- 商品名キーワードで絞り込み検索
- 落札価格の統計情報表示（平均・最高・最低）
- 落札日時順でソート表示

## 技術スタック

- **フロントエンド**: Next.js 15 + React 19 + TypeScript
- **スタイリング**: Tailwind CSS
- **スクレイピング**: Cheerio
- **ランタイム**: Node.js

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開く

## 使い方

1. ヤフオクで対象の出品者の評価ページを開く
   - 例: `https://auctions.yahoo.co.jp/jp/show/rating?auc_user_id=xxxxx&role=seller`

2. そのURLをコピーしてアプリの「出品者ページURL」欄に貼り付け

3. 検索したい商品名キーワードを入力（任意）
   - 例: "ビカクシダ"
   - 空欄の場合は全商品を表示

4. 「検索」ボタンをクリック

5. 検索結果を確認
   - 統計情報（平均・最高・最低価格）
   - 商品一覧（商品名・落札価格・落札日時・詳細リンク）

## 注意事項

- スクレイピングはYahooのサーバーに負荷をかけないよう、適切なレート制限を設けています
- 大量の商品がある場合、検索に時間がかかることがあります
- Yahoo側のHTML構造が変更された場合、動作しなくなる可能性があります
- 個人利用の範囲でご使用ください
